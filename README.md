# 智能填表助手浏览器插件

## 项目概述

这是一个融合了原有搜索助手和表单填写功能的全新浏览器插件，实现了完整的ReAct智能体架构，能够在浏览器环境中自动填写各种表单。

## 项目特点

- **完整的ReAct架构**: 实现了思考-行动-观察的智能循环
- **多模型支持**: 支持DeepSeek等大语言模型
- **浏览器集成**: 无缝集成到Chrome扩展中
- **智能表单解析**: 自动识别和分析表单字段
- **上下文感知**: 填写时考虑已填写字段的上下文
- **用户友好界面**: 直观的侧边栏操作界面
- **灵活工具系统**: 支持多种工具扩展

## 功能特性

### AI驱动的表单填写
- 智能分析表单字段需求
- 自动生成合适的填写内容
- 支持学术搜索获取准确信息
- 上下文感知的一致性保证

### 用户交互
- AI推荐与手动输入结合
- 实时进度跟踪
- 填写统计信息
- 历史记录管理

### 技术能力
- 自动表单解析
- DOM元素定位
- 多种输入类型支持
- 跨站内容提取

## 安装和使用

### 开发者模式安装
1. 打开Chrome浏览器
2. 进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `form-filling-assistant` 文件夹

### 配置API密钥
1. 点击扩展图标打开侧边栏
2. 点击"AI设置"按钮
3. 输入DeepSeek API密钥
4. 保存设置

### 使用流程
1. 访问需要填写的表单页面
2. 点击扩展图标打开侧边栏
3. 点击"使用当前页面"获取URL
4. 点击"解析表单"分析表单结构
5. 点击"开始填表"启动AI填写流程
6. 根据需要选择AI推荐或手动输入

## 技术架构

### 核心组件
- **ReAct智能体**: 实现推理和行动循环
- **工具执行器**: 管理和执行各种工具
- **表单解析器**: 分析HTML表单结构
- **浏览器控制器**: 管理浏览器交互
- **内容脚本**: 与目标网页直接交互

### 通信架构
```
侧边栏界面 ←→ 背景脚本 ←→ 内容脚本 ←→ 目标网页
     ↑              ↑           ↑
   LLM API      消息路由     DOM操作
```

## 文件结构

```
form-filling-assistant/
├── manifest.json                 # 扩展配置
├── background.js                 # 背景服务
├── sidebar.html                  # 侧边栏界面
├── sidebar.js                    # 侧边栏逻辑
├── styles.css                    # 样式文件
├── content_script.js             # 内容脚本
├── icons/                        # 扩展图标
└── js/                           # JavaScript模块
    ├── llm_client.js             # LLM客户端
    ├── react_agent.js            # ReAct智能体
    ├── form_agent.js             # 表单填写智能体
    ├── form_parser.js            # 表单解析器
    ├── form_tools.js             # 表单工具
    ├── tool_executor.js          # 工具执行器
    ├── tools.js                  # 工具集合
    └── browser_controller.js     # 浏览器控制器
```

## 扩展开发

### 添加新工具
1. 在 `js/tools.js` 中创建新工具类
2. 在 `js/tool_executor.js` 中注册工具
3. 实现相应的功能逻辑

### 自定义字段类型
1. 在 `js/form_parser.js` 中添加字段类型识别
2. 在 `js/form_tools.js` 中添加填写逻辑

## 图标文件说明

项目中的图标文件目前是占位符文件。要正常使用扩展，请：

1. 准备三个PNG格式的图标文件：
   - icon16.png (16x16像素)
   - icon48.png (48x48像素)
   - icon128.png (128x128像素)
2. 替换 `icons/` 文件夹中的同名文件
